2. Introducere

2.1 Scopul documentului

Scopul acestui document este sa prezinte o descriere detaliata a aplicatiei Web Paste It bazata pe proiectul propus PIt (Paste It). Acesta va explica scopul si functionalitatile aplicatiei, interfata aplicatiei si constrangerile acesteia. Acest document este destinat potentialilor utilizatori ai aplicatiei.

2.2 Conventiile documentului

Acest document a fost create pe baza modelului IEEE System Requirement Specification Documents.

2.3 Audienta tinta

Persoane obisnuite care vor sa distribuie cod online in diferite limbaje de programare, intr-un mediu creat in acest sens (ex. elevi/studenti, programatori, cursanti sau persoanele interesate de programare)

2.4 Scopul Produsului

Paste It a fost dezvoltat cu scopul de a ajuta si incuraja oamenii sa invete programare si sa distribuie cod cu alti utilizatori intr-un mediu eficient si organizat. De asemenea, el poate fi folosit pentru colaborarea asupra codului sursa intr-un mod flexibil si personalizabil.

3. Descriere generală

3.1 Descrierea produsului

Paste It este o aplicatie Web care realizeaza managementul postarilor ce contin cod sursa pentru diferite limbaje de programare. Aplicatia este destinata dezvoltatorilor si pasionatilor de programare care doresc sa impartaseasca si sa colaboreze asupra codului sursa.

3.2 Funcționalitățile produsului

Aplicatia ofera posibilitatea de a cauta postari dupa nume (intr-un search bar) sau categorie cat si de a cauta alti utilizatori. PIt permite vizitatorilor website-ului sa caute postari sau sa distribuie cod fara a avea un cont creat sau sa se inregistreze/logheze. Utilizatorii logati pot, in plus, selecta alti utilizatori cu drepturi de modificare sau pot avea acces la functionalitatile "burn after read" sau "password protected". De asemenea, aplicatia permite limitarea duratei de pastrare a postarilor si sintax highlighting pentru codul sursa.

Home Page(guest)

  • My account: utilizatorul este redirectionat catre pagina de login (unde exista si optiunea de inregistrare)
  • Search section: utilizatorul poate cauta o postare cu ajutorul unui search bar.
  • Products: utilizatorul poate selecta o postare pentru a o accesa si vedea.
  • Upload Data: utilizatorul poate face o postare ce va fi valabila pe aplicatie 30 de zile.

Home Page(logged-in user)

  • Account settings: utilizatorul este redirectionat catre pagina unde isi poate edita datele contului.
  • Search section: utilizatorul poate cauta o postare cu ajutorul unui search bar.
  • Uploads: utilizatorul poate selecta o postare anterioara pentru a o vedea.
  • Upload Data: utilizatorul poate face o postare si va fi redirectionat pe pagina specifica de upload.

SignUp page

  • Create account form: utilizatorul creeaza un cont nou folosind date valide, iar apoi este redirectionat catre home page (logged-in user)
  • Login: utilizatorul va fi redirectionat catre pagina de logare
  • Don't have account: utilizatorul este redirectionat catre home page (guest)

Welcome page

  • Login form: utilizatorul se conecteaza la un cont deja existent folosind date valide, iar apoi este redirectionat catre home page (logged-in user)
  • Create one: utilizatorul este redirectionat catre pagina de SignUp page
  • Don't have account: utilizatorul este redirectionat catre home page (guest)

Profile page

  • Change icon: utilizatorul isi poate alege o fotografie ce va fi atasata contului personal
  • Edit profile: utilizatorul va fi redirectionat catre pagina de editare a datelor contului
  • Sign out: utilizatorul se va deconecta de la cont si va fi redirectionat catre Welcome page

3.3 Categorii de utilizatori

  • Guest: simplu visitator al site-ului ce nu poate accesa toate facilitatile site-ului. Acesta poate vedea postari existente pe platforma si crea postari valabile maxim 30 de zile.
  • User: utilizator cu un cont creat si stocat in baza de date ce poate accesa toate facilitatile site-ului.

3.4 Mediu de operare

Fiind o aplicatie Web, Paste It poate fi utilizata in orice browser ce rulează HTML, CSS, netinand cont de o versiune anume a sistemului de operare a device-ului pe care ruleaza aplicatia.

3.5 Design și constrângeri de implementare

Aplicatia Paste It va fi dezvoltata in Node.js pentru partea de backend si utilizand html, css si javascript pentru partea de frontend.

4. Cerinte externe

4.1 Interfață software

Paste It dispune de conectare la o baza de date relationala: MySQL, pentru a stoca datele utilizatorilor cat si a postarilor. Cerintele minime software ale aplicatiei sunt un browser functional si conexiune la internet.

4.2 Interfață de comunicare

La nivelul comunicării dintre clienti si server se utilizeaza protocolul HTTP.

5. Cerințe funcționale

5.1 Home page (guest):

Vizitatorul siteului poate vizualiza si crea postari, poate efectua o cautare utilizand bara de cautare sau accesa butonul MyAccount pentru a fi redirectionat catre pagina de logare.

5.2 Home page (logged-in user):

Utilizatorul conectat poate vizualiza si cauta postari utilizand bara de cautare si poate accesa toate celelalte pagini si functionalitati ale aplicatiei.

5.3 SignUp page:

Vizitatorul aplicatiiei Web poate sa isi creeze un cont, completand formularul necesar, dupa care va fi conectat la nou cont si redirectionat catre pagina principala (Home Page).

5.4 Login page:

Vizitatorul, care are deja un cont, poate completa informatiile valide in fomularul de conectare, dupa aceasta va fi redirectionat catre pagina principala (Home Page).

6. Cerinte nonfunctionale

6.1 Cerințe de siguranță

O problema legata de siguranta ar putea fi pierderea informatiilor din baza de date folosita.

6.2 Cerințe de securitate

Aplicatia nu are cerinte de securitate, iar orice utilizator o poate folosi fara a avea posibilitatea de a beneficia de anumite privilegii. (nu exista varinta de cont premium).
